> WebApp for travellers around the world to describe their itineraries and help other people decide if a city is the perfect place to enjoy vacation or not.

### ✨ [Demo](https://mytinerary-budtke.herokuapp.com/)

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```
## Technologies
```
MongoDB / Mongoose
Express
React
Node.js
Redux
JWT / Passport
Joi
```
